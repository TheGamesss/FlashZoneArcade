Loading the browser runtime and checking whether this AIR build can start on the web.

The game is injected directly into the box above. This shell keeps the original SWFs untouched and is ready for static hosting on Netlify or Render.
In case you aren't familiar with the controls, here's the default keyboard controls:
  • Move: Left and Right Arrow Keys
  • Jump: S
  • Duck: Down Arrow
  • Hold down while running to slide, while moving backwards to roll, or while landing to slide boost.
  • Stomp or slide into a baddie to stun them, run into a stunned baddie to kick.
